# HPR Inventory Demo

Standalone demo scene and assets for com.hpr.inventory.

## Scene
- `Packages/com.hpr.inventory/Demo/InventoryDemo.unity`

## Included assets
- `HealthPotion.asset`
- `RifleAmmo.asset`
- `SilverKey.asset`

## What it shows
- configuring an `InventoryComponent` from `ItemData`
- adding and removing stackable items by id
- observing `ItemAdded` and `ItemRemoved` events
- resetting back to starting quantities
