Intermediate report - 2026-03-24

Current state
- Unity workspace is active under `unity/projects/fps_demo` with reusable packages under `unity/packages`.
- The FPS runtime is being extended without replacing the working gameplay systems.
- The current pass adds behavior-driven weapons/enemies, a central event bus, persistent bootstrap/gameplay scenes, asset classification, and automated validation.
- Imported Asset Store content remains local-only and is still excluded from Git tracking.

Architecture extension status
- `WeaponData` now carries `FireModeType`; runtime fire behavior is delegated through `IFireMode` implementations.
- `EnemyData` now carries `AIType`; runtime AI behavior is delegated through `IEnemyBehavior` implementations.
- Added central event infrastructure with `GameEvent`, `EventManager`, and gameplay events for damage, pickups, firing, and enemy deaths.
- Damage, impacts, ammo pickup handling, and enemy death propagation now go through the event bus instead of direct cross-system field access.
- Added `AssetMetadata` and `AssetRegistry` so imported/local assets can be classified without hardwiring them into gameplay code.
- Scene setup now uses `Bootstrap` -> `Gameplay`, with persistent editable scene files and root scene containers for environment, enemies, and props.
- Added `GameStateValidator` and extended the smoke test to validate state changes instead of only checking that the game launches.

Validation completed
- Unity batch build succeeded with `SceneBootstrap.BuildLinux`.
- Linux player smoke test succeeded from the built player with `-smoketest`.
- Verified smoke path covers session start, player damage, weapon firing, inventory pickup, door interaction, enemy kill, save, load, and clean exit.

Local-only asset workflow
- Asset Store package downloads and import metadata remain local-only.
- Imported asset folders and generated imported-asset metadata stay ignored in `.gitignore`.
- Local downloaded/imported content can be used for integration/testing without being committed.

Immediate next focus
